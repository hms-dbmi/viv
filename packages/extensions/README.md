# @vivjs/extensions

> A collection of Deck.gl [`LayerExtension`](https://deck.gl/docs/developer-guide/custom-layers/layer-extensions#layerextension-interface) modules for `@vivjs/layers`

## Installation

```sh
npm install @vivjs/extensions
```
